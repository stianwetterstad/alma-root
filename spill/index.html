Spillmeny

🎮 Spill

Velg et spill! Alle er laget for å fungere bra på iPad (touch/knapper), og også på PC med tastatur.

🐍 Snake

Samle epler 🍎 og unngå å krasje i deg selv. Klassikeren!

Touch: sveip ←↑→↓ Raskt
Spill Snake → Åpne

🧱 Tetris

Roter brikker og lag linjer. Øker i fart etter hvert.

←→ flytt ↑ roter Space dropp Touch-knapper
Spill Tetris → Åpne

👾 Space Invaders

Skyt invaderne før de når bunnen. Pass på livene!

←→ flytt Space skyt Touch-knapper
Spill Space Invaders → Åpne

🧱🏓 Breakout

Dra fingeren for å styre plata og knus alle klossene!

iPad: dra for å styre Enkelt Korte runder
Spill Breakout → Åpne

🧠 Memory

Finn like par! Rolig og supert på touch.

Trykk på kort Barnevennlig Enkel/Vanskelig
Spill Memory → Åpne

🔢 2048

Sveip for å slå sammen tall. Klarer du 2048?

iPad: sveip ←↑→↓ Hjernetrim
Spill 2048 → Åpne
Tips: På iPad kan det hjelpe å legge siden til på hjemskjermen (Del → “Legg til på Hjem-skjerm”) for en mer “app”-følelse.
